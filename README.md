# KINISIYA - منصة بوت واتساب المتطور

## 📋 عن المنصة
kinisiya هي منصة متكاملة لبوت واتساب المتطور، تقدم خدمات إدارة المجموعات والمميزات المتقدمة.

## 📑 الصفحات
- **الرئيسية** (`index.html`) - الصفحة التعريفية الرئيسية
- **السيرفرات** (`servers.html`) - باقات استئجار السيرفرات
- **Nova Ultra** (`nova-ultra.html`) - صفحة المنتج الرئيسي

## 🎨 التقنيات المستخدمة
- HTML5
- CSS3 (تصميم متجاوب)
- GitHub Pages (للاستضافة)
